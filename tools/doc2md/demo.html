Doc to Markdown — demo

Convert DOCX, XLSX/XLS/ODS, PPTX, EPUB, HTML, CSV/TSV, JSON, XML, TXT or a ZIP of them into Markdown — all in browser memory, nothing is uploaded. mammoth and SheetJS load from CDNs with SRI pins; PDF is supported on the site build (pdf.js is an ES module, which script-tag SRI can't pin, so this demo omits it).
